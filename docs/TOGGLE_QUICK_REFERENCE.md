## Usage Tips

### Tip 1: Focus Mode
**Goal:** Minimize distractions, focus only on adding tasks
**Action:** 
- Hide All except "Add New Task" (which is always visible)
- Or disable Statistics, Performance, Visualizations, Insights

### Tip 2: Analysis Mode
**Goal:** Deep dive into productivity patterns
**Action:**
- Enable: Visualizations, Performance, ML Insights
- Disable: Statistics, Goals, Import, Task Management

### Tip 3: Quick Task Mode
**Goal:** Rapid task completion and tracking
**Action:**
- Enable: Task Management, Goals
- Disable: All others

### Tip 4: Full Dashboard
**Goal:** See everything at once
**Action:**
- Click "âœ… Show All" button

### Tip 5: Minimal View
**Goal:** Clean slate, reduce information overload
**Action:**
- Click "âŒ Hide All" button

## Keyboard Workflow (Manual)

1. Click sidebar to expand (if collapsed)
2. Scroll to "âš™ï¸ Display Settings"
3. Check/uncheck desired sections
4. Sections update immediately
5. Use Quick Actions for bulk changes

## What Gets Hidden

When you disable a section:
- **Statistics**: All metrics, category pie chart, time trend
- **Visualizations**: All three tabs (Basic, Productivity, Deep Insights)
- **Performance**: Performance dashboard, score trend, priority chart
- **Goals**: Goal setting inputs and progress bars
- **Task Management**: Recent tasks list and filters
- **ML Insights**: Peak hours, weekly summary, workload balance, ML analysis
- **Import Data**: CSV upload interface

## What Always Stays

These elements are ALWAYS visible:
- App header and branding
- Daily motivational quote
- Main productivity score gauge
- Filter & Search section
- Add New Task form
- Data validation alerts

## Technical Notes

- Toggle states persist during your session
- Refreshing the page resets all toggles to default (enabled)
- No data is lost when hiding sections
- Performance may improve with fewer visible sections
- Mobile-friendly responsive design maintained

---

**Pro Tip:** Start with "Show All" and gradually disable sections you don't need for your current workflow!
